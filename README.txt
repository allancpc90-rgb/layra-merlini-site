# Site Layra Merlini Advocacia

## Como usar no GitHub e Vercel

1. Envie todos estes arquivos para um repositório no GitHub.
2. Troque o número do WhatsApp em `index.html`:
   - procure por `5533999999999`
   - substitua pelo seu número real, sem espaços e sem traços
3. Na Vercel, clique em **New Project**.
4. Escolha o repositório.
5. Clique em **Deploy**.

- `index.html`
- `styles.css`
- `script.js`
